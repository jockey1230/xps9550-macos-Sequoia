## 注意事项

### WiFi 和蓝牙驱动
如需驱动 WiFi 和蓝牙，请按以下步骤操作：
1. 下载最新版 OpenCore Legacy Patcher
2. 运行 OpenCore Legacy Patcher
3. 点击 Post-Install Root Patch
4. 点击 Start Root Patching
5. 等待运行完成
6. 重启系统即可完成驱动
